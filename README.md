# Notes App

A simple Notes App built using FastAPI and SQLite.

## Features

- Add Notes
- Delete Notes
- SQLite Database
- FastAPI Backend
- HTML + CSS UI

## Installation

```bash
pip install fastapi uvicorn jinja2 python-multipart
```

## Run Project

```bash
python -m uvicorn main:app --reload --port 8001
```

## Open in Browser

http://127.0.0.1:8001

## Tech Stack

- FastAPI
- SQLite
- HTML
- CSS
- Jinja2
