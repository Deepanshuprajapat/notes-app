# Notes App

A simple Notes App built using FastAPI and SQLite.

## Features

* Add Notes
* Delete Notes
* SQLite Database
* FastAPI Backend
* HTML + CSS UI

## Tech Stack

* Python
* FastAPI
* SQLite
* HTML
* CSS
* Jinja2

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/notes-app.git
```

Go to project folder:

```bash
cd notes-app
```

Install dependencies:

```bash
pip install fastapi uvicorn jinja2 python-multipart
```

## Run the Project

```bash
python -m uvicorn main:app --reload --port 8001
```

## Open in Browser

```text
http://127.0.0.1:8001
```
